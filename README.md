---

## About

Single-file HTML/CSS/JS portfolio. No build step, no npm, no framework — just open and deploy.

**Sections:** Experience · Projects · Skills · Roadmap · Contact

---

## Tech

- Pure HTML5 + CSS3 + vanilla JS
- Google Fonts (Playfair Display + DM Sans + DM Mono)
- Intersection Observer for scroll animations
- No dependencies

---

## Run locally

```bash
# Option 1 — just open it
open index.html

# Option 2 — local server (avoids CORS on fonts)
python3 -m http.server 3000
# then visit http://localhost:3000
```

## Deploy

Drag the folder onto [vercel.com](https://vercel.com) or [netlify.com](https://netlify.com). Done in 60 seconds.

---

## Customise

All design tokens are CSS variables at the top of `<style>`. Change `--violet`, `--amber`, `--paper` to retheme the entire site in 3 lines.
